Combobox загружаемый из внешнего файла.

Часто возникают ситуации, когда элемент select с один и тем же содержимым
используется на нескольких страницах. 
Чтобы избежать copy/paste можно загружать элемент из внешнего файла.
 

Вот работающий элемент




Для того, чтобы его сделать, надо создать один или несколько файлов с наполнением 
аналогичным стандартному наполнению тега select, т.е. тегами option. 
Файл должен иметь кодировку utf-8.
В самом документе в тегах option в атрибуте value указывается путь к файлу.
В тексте тега option может указываться заголовок списка(заготовка на будущее).
