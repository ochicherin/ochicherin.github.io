Основные принципы построения сайта.

Звучит конечно очень амбициозно, но попытаюсь объяснить.
  1. Для создания сайта используется текстовой редактор.
  2. У страницы есть рамка, которая автоматически обрамляет страницу.
  3. В каждой папке может существовать своя рамка. Если рамки нет, она наследуется из родительской папки.
  4. Сама страница является рамкой для элементов находящихся на других страница.
  5. Страница может быть подгружена не полностью, а только определенная секция.
  6. Навигация. Единая карта сайта плюс rss лента.
