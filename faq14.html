Обновление через веб интерфейс.

Распакуйте содержимое архива в отдельную папку.

1. Меню "Management"->"Upgrade Firmware"
2. Нажать "Browse", выбрать файл linux-1.4.bin
3. Нажать "Upload", по окончании обновления нажать "Ok"
4. Дождаться окончания перезагрузки устройства и еще раз войти в веб-интерфейс роутера
5. Повторить шаги 1-3 для файла webpages-vpn.bin

6. Меню "Management"->"Save/Reload Settings"
7. Нажать "Browse", выбрать файл config-vpn-ogf.bin
8. Нажать "Upload", по окончании обновления нажать "Ok"

9. После окончания обновления рекомендуется очистить кэш браузера



Прошивка при помощи tftp

Устанавливаем на компьютере адрес 192.168.1.1/255.255.255.0
Выключаем роутер.
Нажимаем и держим reset.
Включаем питание.
Ждем 6 секунд.

Роутер станет доступен по адресу 192.168.1.6
При этом ping не идет.

Загружаем прошивку командами

tftp -i 192.168.1.6 put webpages-vpn-eng15.bin
tftp -i 192.168.1.6 put linux-1.4.2Q_eng16.bin


Конфиг лучше не заливать. Может плохо кончиться.
